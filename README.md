# Aurum

DeFi Ghost Tracker — AAVE V3 + Morpho Blue monitor with PT IRS tax tracking.

**Live app:** https://danyundercrypto.github.io/aurum/

## Features

- 🏦 AAVE V3 multi-chain monitoring (Ethereum, Base, Arbitrum, Optimism, Linea, Polygon, Scroll, Gnosis)
- 🔵 Morpho Blue positions on-chain
- 💳 Crypto card import (ether.fi, MetaMask, Tria + universal adapter)
- 📊 365-day exemption tracker (PT IRS — Lei 24-D/2022)
- 📑 Tax report exports (Anexo G / G1 / Cashbacks)
- 🌌 Real-time HF alerts via on-chain calls

## Privacy

Self-custodial. All data stays in your browser (localStorage). No server.
